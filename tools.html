在线开发工具箱 Pro

一站式前端工具网站，支持二维码、JSON、Base64、URL Encode/Decode、时间戳秒/毫秒转换、MD5 / SHA-256、文本大小写、JWT 解析、正则测试、UUID 生成等常用工具。所有处理均在浏览器本地完成。所有代码由AI生成。

二维码生成

输入文本、链接或任意内容后立即生成二维码。短文本默认使用较高纠错与紧凑尺寸，较长内容会自动放宽尺寸并适度降低纠错级别，避免过度压缩影响扫码。

自适应尺寸 + 纠错
  • 120 字以内：220×220，H
  • 121-260 字：280×280，H
  • 261-500 字：340×340，Q
  • 500 字以上：420×420，M

JSON 格式化 / 压缩

可对 JSON 文本进行美化、压缩，并在格式错误时给出提示。

Base64 编码 / 解码

支持普通文本进行 Base64 编码与解码。

URL Encode / Decode

用于对 URL 参数进行编码和解码，适合 querystring 与路径片段处理。

时间戳转换

支持 Unix 时间戳与本地日期时间互转，可一键切换秒级 / 毫秒级。

MD5 / SHA-256 哈希

对输入文本生成 MD5 或 SHA-256 摘要值，适合校验与快速比对。

文本大小写转换

JWT 解析

解析 JWT 的 Header 和 Payload，显示签名部分，并把过期时间转为可读日期。

正则测试

输入正则表达式和测试文本，支持 g / i / m 标志，快速查看匹配结果。
